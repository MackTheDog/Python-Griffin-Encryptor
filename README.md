# Python-Griffin-Encryptor
An easy to use drop in and encrypt app. Robust and simple to use for Message, videos, images and files.

Purpose
Create an encyptor using AESGCM, Argon2id 256, padding for small files, 32 byte salt , Public proof (Robust),
provides zero information to an attacker who has an encrypted .grn.txt package requiring encrypted headers and no
means to determine how many files are in the package nor how big the individual files are. GPE can encrypt anything typed into the window as well as anything dropped into the window.  Decryption allows the user to place the resultant files, mesaage, images and videos wherever the user selects. Simple interface that includes the option of shredding of the source files if the user chooses. Written in Python.
