# Python-Griffin-Encryptor
An easy to use drop in and encrypt app. Robust and simple to use for Message, videos, images and files.

Purpose
Create an encyptor using AESGCM, Argon2id 256, padding for small files, 32 byte salt , Public proof (Robust),
provides zero information to an attacker who has an encrypted .grn package requiring encrypted headers and no
means to determine how many files are in the package nor how big the individual files are. Ideally GPE should 
also have the ability to encrypt anything typed into the window as well as anything dropped into the window
including videos, images and files and any text written into the window itself. There should be at least
8 characters for the password. Decryption should allow the user to place the reultant files, mesaage, images and
videos wherever the user selects.
